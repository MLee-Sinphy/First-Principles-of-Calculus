# First Principles of Calculus

A rigorous and interactive introduction to differential and integral calculus,
developed from first principles with mathematical precision, historical
context, formal proofs, and modern web technologies.

---

## Overview

**First Principles of Calculus** is an open educational project whose goal is
to reconstruct differential and integral calculus from its logical foundations.

Rather than presenting isolated computational techniques, the project develops
each concept from carefully stated assumptions, precise definitions, rigorous
proofs, historical motivation, and interactive visualizations.

The long-term objective is to build a modern digital textbook combining the
rigor of classical mathematical analysis with the interactivity made possible
by contemporary web technologies.

## Philosophy

This project follows a simple principle:

> Mathematics should be understood before it is computed.

Every topic is developed according to the following structure:

1. Historical motivation
2. Conceptual discussion
3. Formal definitions
4. Theorems
5. Proofs
6. Examples
7. Exercises
8. Interactive visualizations

The objective is to combine intuition with mathematical rigor instead of treating them as opposing approaches.
